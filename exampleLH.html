There are 6 different levels of headers, each having a different size for text.

Level 1: Tables

A table displays information just like any spreadsheet program. It's border and style can be changed as well.

Images can also be added to a webpage, either using a url or the address to a locally stored image. They can also be sized up or down to fit the page.

Level 3: Ordered List

An ordered list shows different elements in ascending order with a slight indentation from the left side of the screen.

  1. List item 1
  2. List item 2
  3. List item 3

Level 4: Unordered List

An unordered list works the same as an ordered list but uses bullet points instead of numbers.

  • List item 1
  • List item 2
  • List item 3
Level 5: Page Spaces using <br>

<br> is used to add breaks in a webpage, since spaces don't translate from the html file to the website displayed.
